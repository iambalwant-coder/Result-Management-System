This project is a simple Student Management System built with Django. It allows users to manage student records efficiently through a web interface.

Project Description üìù

The Student Management System provides functionalities to:

  • Add Students: Users can input student details, including ID, name, and marks in various subjects.
  • Edit Student Records: Users can update existing student information and recalculate results based on updated marks.
  • Delete Students: Users can remove student records from the system.
  • View Student Details: Users can view detailed information about each student, including their total marks and result status.
  • Filter Students: Users can filter the list of students based on their names and result status (PASS/FAIL).

This system is designed to help educational institutions keep track of student performance and manage records effectively.
